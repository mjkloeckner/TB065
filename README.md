# Trabajo Práctico Especial

## Primera Parte

En esta primera parte del trabajo practico se pide graficar dos señales de
audio en formato 'WAV' en el dominio del tiempo

Para realizar los graficos se utiliza el lenguaje de programación
[python](https://www.python.org/) junto con las librerias
[scipy](https://docs.scipy.org/doc/scipy/index.html),
[matplotlib](https://matplotlib.org/) y
[numpy](https://numpy.org/doc/stable/index.html)

En particular se hace uso de las siguientes funciones:

* [scipy.io.wavfile.read](https://docs.scipy.org/doc/scipy/reference/generated/scipy.io.wavfile.read.html)
* [numpy.arange](https://numpy.org/doc/stable/reference/generated/numpy.arange.html)

## Segunda Parte

En la segunda parte del trabajo práctico se realiza un analisis en el dominio de
frecuencia de los mismos archivos que en la primera parte.

Se utiliza el mismo entorno que en la primera parte para realizar los graficos.
